[概要]
iGlyphTrainerは、IngressのGlyph Hackの練習が目的のiOS用ゲームアプリです。
Glyph Hackを成功させる事で、Ingressのゲームを有利に進める事ができます。

出題するセンテンスのグリフ数を調整可能です(2から5)ので、
徐々にGlyph Hackに慣れていく事ができます。
また、出題中と回答中にグリフの名前を表示する事が出来るので、
グリフの名前を無理なく覚えられます。

ゲーム実施に際して、ネットワークアクセスは発生しませんので、
ネット接続が難しい場所でも遊べます。広告が表示される事もありません。

本ソフトウェアは、無料のオープンソースソフトウェアで、GNU Public License Version 2.1
の下配布されています。ソースコードは、github (https://github.com/steelwheels/iGlyphTrainer)
から入手できます。

使い方の詳細については、ユーザーズマニュアル
  https://sites.google.com/site/steelwheelsproject/home/products/iglyphhacktrainer
を参照してください。

ご注意:　Ingressは、Niantic Lab.の製品です。本ソフトウェアは、Niantic Lab.やgoogleとは無関係です。

[比較]
競合するアプリは、"GHTranslator"のみです。
こちらもよく出来たソフトですし、コンセプトには共通点があるのですが、
* UIの反応速度
* ドキュメント(およびドキュメントへのリンク)
点について、本アプリの方が勝っていると思われます。

[セールス]
Ingressのプレーヤは世界中に多数おられます(1000万ダウンロードを超えています)。
Glyph Hackは、Ingressの有利なアイテムを入手する際に必要なミニゲームですので、
その練習には相当数のプレーヤが興味があると思われます。

[Desription]
iGlyphTrainer is an iOS application to train Ingress Glyph Hacking.
If you get high score at the glyph hacking in Ingress,
you can use the acquired item to advantageously develop a game.

The number of glyphs in a sentence can be controlled (from 2 to 5).
So you can get used to hack the glyph.
And you can control whether the glyph name is displayed at the question/answer
sequence or not. It helps you that you memorize the glyph name.

You can get the source from github (https://github.com/steelwheels/iGlyphTrainer).
